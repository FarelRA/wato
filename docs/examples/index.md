# Examples

This section is the practical side of the `wato` docs.

Use these guides when you want something concrete to copy, adapt, and run.

## Example sets

- `workflow-patterns.md` - reusable workflow building blocks and end-to-end workflow examples
- `webhook-recipes.md` - webhook setup patterns, payload handling, and replay flows
- `api-cli-recipes.md` - copy-paste API and CLI recipes for daily operations
- `use-cases.md` - real-world automation scenarios and how to model them in `wato`

## Suggested path

1. Start with `workflow-patterns.md`
2. Add notifications with `webhook-recipes.md`
3. Operationalize with `api-cli-recipes.md`
4. Adapt one of the production-style ideas from `use-cases.md`
